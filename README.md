# jsonpath-debs
Build Debian Package for jsonpath

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)
- libmaven-bundle-plugin-java
- libslf4j-java
- libjackson-json-java
- [libjson-smart-java](https://github.com/yadickson/json-smart-debs)

**Tested**

- Debian wheezy
- Debian jessie
